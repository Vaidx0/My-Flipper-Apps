# My-Flipper-Apps

## Info

* Device: Flipper Zero
* Author: Amau_Zero

## Apps

* **car_identifier.fap**

  * Car keyfob scanner

* **doorbell_scanner.fap**

  * Doorbell scanner

* **interphone_db.fap**

  * Intercom / interphone database

* **password_cracker.fap**

  * Password strength testing

## Usage

* Copy `.fap` files to Flipper Zero
* Launch from Apps menu
